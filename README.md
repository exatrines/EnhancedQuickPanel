# Enhanced Quick Panel

A customizable quick panel overlay for FFXIV ([Dalamud](https://github.com/goatcorp/Dalamud) plugin).

Build your own panels of actions, items, macros, and text commands, spread them across multiple pages, and style them however you like. Optionally replace the game's native quick panel with the overlay.

## Repository
```
https://raw.githubusercontent.com/exatrines/DalamudPlugins/refs/heads/main/pluginmaster.json
```

## Quick start

| Command | Action |
| --- | --- |
| `/enhancedquickpanel` or `/eqp` | Toggle the overlay |
| `/enhancedquickpanel settings` or `/eqp settings` | Toggle the settings window |
